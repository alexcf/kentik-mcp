# 2026-01-05 to 2026-01-11

Source: https://kb.kentik.com/docs/2026-01-05-to-2026-01-11

---

## AI / Insights

### Features

* **AI Advisor: Device Command Tool** — A new Device Command tool is now available in AI Advisor. A new "Command Access" tab in AI settings lets you control usage of the config tool and configure command ACLs. The Configs device settings UI has been updated to better distinguish between configuration fetch and running commands.
* **AI Advisor: Tool Run Approval** — AI Advisor tools can now require user approval before execution. You can approve a tool once, approve it for the entire session, or reject it.
* **AI Advisor: Input Validation** — AI Advisor now validates IP addresses and CIDR blocks entered into the Interface Tool, ensuring valid input and prompting for corrections if needed.
* **AI Advisor A2A Chat Interface** — A new Streamlit-based web chat interface for the AI Advisor A2A agent provides an intuitive conversational UI with real-time reasoning visibility, conversation context management, and message history.

## Alerting

### Features

* **Custom Email Notification Templates** — You can now customize email notification subject lines and body content using Go template syntax. A new template editing experience includes a split-pane interface with live HTML preview, real-time error reporting, intelligent autocomplete for template data paths, and persistent editor preferences.
* **Improved Alert Policy Error Status** — Alert policy error status now properly propagates from parent policies to subpolicies, providing clearer visibility into policy health.

### Bug Fixes

* **Mitigations API Fix** — Fixed a missing prefix in the public mitigations API endpoint, restoring proper functionality.

## Cloud

### Features

* **Cloud API Call Monitoring** — New tracking and monitoring capabilities for cloud API call metrics and error patterns.

### Bug Fixes

* **AWS Cloud Enrichment Fix** — Fixed missing AWS cloud resource enrichment (ENI metadata, service information) for network flows in certain VPCs, restoring proper identification and classification of traffic from AWS resources like EC2 instances, RDS, ALBs, and ElastiCache.

## Core

### Features

* **Notification Channel Editing Redesign** — Notification channels can now be created and edited on dedicated, URL-addressable pages instead of modal pop-ups, providing a more streamlined editing experience.
* **User Attribute Flow Enrichment** — User Attribute Flow Enrichment is now generally available (feature flag removed).
* **Interface Details Performance** — Faster loading of interface details drawer on device inventory pages through query optimization.
* **Interface Streaming Performance** — A new high-performance gRPC streaming endpoint for retrieving interface data delivers 1.3x–3.4x faster performance with smaller batched payloads and improved handling of non-UTF-8 characters.
* **Extreme Networks Device Support** — Added support for Extreme Networks device models (including BlackDiamond, Summit, and Altitude series) for proper SNMP device identification and monitoring.
* **Chatsworth eConnect PDU MIB Support** — Added support for Chatsworth eConnect PDU MIBs (CPI MIBs) for SNMP monitoring, split into three profiles (legacy, unity, secureArray).
* **Query Engine Performance** — Increased worker concurrency in the KDE query system to 75% of available CPU cores, improving query throughput and performance.
* **DNS Analytics Improvements** — Updated DNS Analytics with more friendly dimension naming, additional enrichment fields, geolocation enrichment for DNS answer IPs, and support for Telia TV streaming service classification. DNS analytics now properly filters out IP addresses outside customer-specified Internal IP ranges.
* **Data Explorer: JSON Table Format** — Added json-table as a new data explorer format option for viewing and exporting query data.
* **Versa Application Custom Dimensions** — Added support for Versa Application custom dimensions (Application Name, Family, Subfamily, Productivity, Risk), enabling application-specific network traffic analysis from Versa devices.
* **KMI Agent Global Availability** — The KMI agent service is now available across all 16 control plane regions with intelligent routing that separates UI traffic from agent-to-agent traffic.

### Bug Fixes

* **Device Deletion Fix** — Fixed an issue where devices could not be deleted or archived due to auto-discovered connections incorrectly blocking the operation. Deletion validation now only checks manually-created connections.
* **Dashboard Email Reports Fix** — Fixed an issue where emailed dashboard reports were generating blank PDFs instead of including dashboard content and visualizations.
* **Signup Link Fix** — Fixed signup link functionality on the Login page to properly handle external links.
* **Juniper vMX SNMP Trap Fix** — Resolved an issue where Juniper vMX SNMP Trap data displayed "unknown values" in Data Explorer for BGP-related traps, restoring proper monitoring of BGP state transition events.
* **ktranslate File Tags Fix** — Fixed a bug where duplicate values across different tag columns were being missed, ensuring all values are properly captured.
* **Metrics Service Stability** — Fixed a crash in the metrics ingestion service when companies with expired status or no valid metrics plan sent metric requests.

## Network Monitoring System (NMS)

### Features

* **Device Subtypes Reorganization** — Device Subtypes have been reorganized to separate Dimension and Metric processing, enabling more efficient and specialized data handling.
* **Agent Diagnostics Improvements** — Improved diagnostic tool functionality by reducing noise from unnecessary domain checks, aligning default output directories, and adding timing information for better log correlation.
* **Agent Health Metrics** — The agent can now push metrics data as a health signal, reporting operational health status with a startup grace period before health checks begin.
* **Agent Entropy Metrics** — Added entropy metrics collection capability for monitoring system entropy levels.

### Bug Fixes

* **Transceiver Lane Metrics Fix** — Fixed display of transceiver lane metrics (Output Power, Input Power, Laser Bias Current) by correcting how interface transceiver lane measurements are queried and filtered.

## Synthetics

### Features

* **UDE Synthetics Interface Enhancements** — Improved Synthetics data visualization in the Universal Data Explorer with better data organization, sorting capabilities, time navigation controls (1h, 3h intervals), HTTP Status Code values, Agent Name rendering, PageLoad Task type support, clickable Agent/Test Name links for navigation, and proper distinction between zero values and missing data points.
