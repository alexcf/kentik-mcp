# How to Submit a Feature Request

Source: https://kb.kentik.com/docs/how-to-submit-a-feature-request

---

We highly value customer feedback and are always looking for ways to improve your experience. If you have an idea for a new feature or an enhancement to an existing tool, you can easily submit a feature request directly through the Kentik Portal.

### Important: Browser Security Exceptions

Kentik partners with **Pendo** to manage and track customer feature requests. When you navigate to the feature request portal, Kentik securely passes your authentication via cookies.

If you utilize advanced security options, strict tracking prevention, or ad-blockers in your web browser, the Feature Requests page may fail to load. To ensure the page loads successfully, please configure your browser to allow cookies and cross-site data exceptions for the following domains:

* `kentik.com`
* `pendo.io`

> **Note**: The exact steps to add these exceptions vary by browser. Refer to your specific browser's official support documentation for instructions on managing cookie exceptions.

### Step-by-Step Instructions

![Form for submitting a Kentik support request with fields for summary and details.](https://cdn.us.document360.io/082e25b5-afce-42d4-8f47-70bd3f1d02b7/Images/Documentation/CC-submit-suppor-request.png?sv=2022-11-02&spr=https&st=2026-05-12T09%3A16%3A20Z&se=2026-05-12T09%3A27%3A20Z&sr=c&sp=r&sig=CV%2BNqTUD5KwbcRMyLar2EMkiHpi%2B3u1o6p%2Bv0ZH9XZw%3D)To access the feature request board:

1. Log in to the Kentik Portal.
2. Click the **?** (Help/Support) icon located in the top-right corner of the main nav bar.
3. In the support menu that appears, locate the prompt at the bottom that asks, "What do you wish you could do with Kentik?"
4. Click the **Feature Requests** button.
5. You will be automatically authenticated and redirected to the **feature request dashboard**.

### Using the Feature Request Dashboard

Once you arrive at the dashboard, you can directly interact with the Kentik product team's backlog and roadmap. Here is what you can do:

* ![Kentik dashboard showing recent releases and suggestion submission options.](https://cdn.us.document360.io/082e25b5-afce-42d4-8f47-70bd3f1d02b7/Images/Documentation/CC-feature-request-main.png?sv=2022-11-02&spr=https&st=2026-05-12T09%3A16%3A20Z&se=2026-05-12T09%3A27%3A20Z&sr=c&sp=r&sig=CV%2BNqTUD5KwbcRMyLar2EMkiHpi%2B3u1o6p%2Bv0ZH9XZw%3D)**Make a Suggestion**: Click the **Make a Suggestion** button on the left side of the dashboard to submit your own unique feature idea.
* **Vote on Existing Ideas**: Browse the **Suggested by Others** list to see what other customers are requesting. You can click the **Vote** button next to ideas you support and track their current Status (e.g., Awaiting Feedback, Planned, or Building).
* **Manage Your Priorities**: The **My Priorities** section automatically tracks the ideas you have submitted or voted on. You can use this space to tell our product team which of those requests you want the most.
* **Track New Releases**: Check the **Recently Released** list on your dashboard or click **Releases** in the left-hand navigation menu to see which user-requested features have recently gone live.
* **View the Roadmap**: Click **What's Coming?** in the left-hand navigation to get a sneak peek at upcoming features and updates.
