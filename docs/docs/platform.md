# Platform

Source: https://kb.kentik.com/docs/platform

The **Kentik Platform** is the comprehensive backend infrastructure for the Kentik Portal.

Explore the Platform category in the KB with the Explore Articles side bar, available anytime you view an article.

## **Subcategories**

* **Configuration**
* **Metrics & Dimensions**
* **Kentik Data Engine**
* **Kentik APIs**

## **Platform vs. Portal**

Still confused about the difference between Kentik Platform and Kentik Portal? Here’s the gist:

* **Platform** = The engine (backend infrastructure, APIs, data processing)
* **Portal** = The interface (frontend UI, dashboards, user workflows)
