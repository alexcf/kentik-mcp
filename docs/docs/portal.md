## **Platform vs. Portal**

Still confused about the difference between Kentik Platform and Kentik Portal? Here’s the gist:

* **Platform** = The engine (backend infrastructure, APIs, data processing)
* **Portal** = The interface (frontend UI, dashboards, user workflows)
