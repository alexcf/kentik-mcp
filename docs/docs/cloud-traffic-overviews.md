Kentik's Cloud Traffic Overview pages are specialized **Dashboards** that become active when Kentik begins monitoring resources within a supported cloud provider.

## Supported Cloud Providers

The **Kentik Portal** currently features dedicated overview dashboards for the following providers:

* **AWS Traffic Overview:** Insights into Amazon Web Services traffic flows.
* **GCP Traffic Overview:** Monitoring for Google Cloud Platform environments.
* **Azure Traffic Overview:** Visibility into Microsoft Azure infrastructure.

## Dashboard Capabilities

These panels deliver flow-based insights into cloud-specific traffic, resource utilization, and infrastructure health. Use these dashboards to:

* **Analyze Traffic Patterns:** View flow-based visualizations tailored to each specific provider.
* **Monitor Infrastructure:** Track various aspects of cloud utilization and service health.
* **Optimize Performance:** Identify top-talkers and regional traffic distribution.
