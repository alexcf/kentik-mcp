# 2025-12-29 to 2026-01-04

Source: https://kb.kentik.com/docs/2026-01-04

---

## Core

* **Improved SNMP health check responsiveness**: The SNMP polling health status indicator now reflects current conditions more accurately by reducing the lookback period from 48 hours to 15 minutes.
* **Fixed device deletion logic for auto-discovered connections**: Device deletion prevention now correctly counts only manually defined connections, no longer blocking deletion due to auto-discovered connections.
* **Updated SSO autologin link format**: The SSO autologin toast link has been updated to use the new simplified URL format (`login/sso/{identifier}?autologin=true`).

## Network Monitoring System (NMS)

* **Fixed custom dimension values in file tags**: Resolved a bug where custom dimension values were being missed when the same value appeared in different columns, ensuring all values are now properly processed and displayed.
* **Improved site data retrieval**: Updated to use the new site API for loading extended site information, providing more reliable and current site data with improved error handling.

## Synthetics

* **Improved agent cache management**: Implemented single agent cache ID functionality for better cache management and agent coordination in the synthetics testing platform.
